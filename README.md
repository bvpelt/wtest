# Inleiding

Gebruik de live server om de javascript te starten

Gebaseerd op: https://www.youtube.com/watch?v=GFO_txvwK_c 

Stappen
- Maak een html pagina zodat canvas gevisualiseerd kan worden [index.html](index.html)
- Voeg evt een style toe om visualisatie te sturen [style.css](style.css)
- Maak een script om op de canvas te tekenen [clock.js](clock.js)

Let op: Default is 0,0 de linker bovenhoek.
